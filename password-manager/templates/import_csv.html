CSV Format

Your CSV file should have this format:

"URL","Username","Password"

"https://google.com","user2","pass456"

Requirements

  • Each row should have exactly 3 columns: URL, Username, Password

  • Special characters are allowed except commas
  • URLs can contain any characters
  • Lines starting with # are treated as comments and ignored
